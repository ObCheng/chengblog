AI的导出文件格式：.AI(工程文件) .EPS(矢量图) .TIF(位图) .PDF(稳定)

快捷键：

| 功能     | 快捷键              |
| -------- | ------------------- |
| 快速全屏 | F/TAB               |
| 保存     | CTRL+S/CTRL+SHIFT+S |
| 放缩     | CTRL+[+]/ALT+滑轮   |
| 恢复     | CTRL+0              |
| 抓手工具 | 空格+鼠标拖拽       |
| 撤回     | CTRL+Z/CTRL+SHIFT+Z |
| 标尺     | CTRL+R              |
| 参考线   | CTRL+;              |
| 网格     | CTRL+“              |
| 轮廓模式 | CTRL+Y              |
| 像素模式 | CTRL+ALT+Y          |
| 叠印模式 | CTRL+SHIFT+ALT+Y    |
